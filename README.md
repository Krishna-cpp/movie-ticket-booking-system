# Movie Ticket Booking System

## Overview
A web-based movie ticket booking application built using Angular to simulate a real-world cinema reservation workflow.  
The system allows users to browse movies, select shows, choose seats, and simulate ticket booking.

## Features
- Movie listing interface with show selection
- Seat selection logic for booking simulation
- Ticket confirmation workflow
- Component-based UI structure using Angular
- Organized routing and state handling

## Tech Stack
Angular, TypeScript, HTML, CSS

## How to Run
1. Clone the repository
2. Navigate into the project folder
3. Install dependencies:
   npm install
4. Start development server:
   ng serve
5. Open http://localhost:4200/

## Project Structure
The application follows Angular’s component-based architecture with separate modules for UI, booking logic, and routing.

## Learning Outcome
This project helped me understand Angular fundamentals, component interaction, routing flow, and how booking systems manage user state and UI updates.
