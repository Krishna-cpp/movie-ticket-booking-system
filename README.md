# 🎬 Movie Ticket Booking System

A web-based Movie Ticket Booking System developed using Angular and TypeScript.  
The application allows users to browse movies, view showtimes, select seats, and confirm bookings through an interactive user interface.

--------------------------------------------------

🚀 Features

- Movie catalogue with posters
- Movie search functionality
- Movie detail page
- Showtime selection for theatres
- Cinema-style seat selection
- Unavailable seat indicators
- Booking confirmation ticket
- Angular routing between pages
- Component-based architecture

--------------------------------------------------

🛠 Technologies Used

- Angular
- TypeScript
- HTML
- CSS
- Angular CLI

⚙️ Installation and Setup

1. Clone the repository

git clone https://github.com/Krishna-cpp/movie-ticket-booking-system.git


2. Navigate to the project folder

cd movie-ticket-booking-system


3. Install dependencies

npm install


4. Run the Angular development server

ng serve


5. Open the application in your browser

http://localhost:4200
